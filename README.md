# HR Data Analysis Project

This project focuses on processing and visualizing human resources data to gain insights into workforce demographics and trends.

## Project Components

*   **SQL Script (`Mini Project 1.csv` / SQL format):** Contains the raw data and/or the queries used to clean and transform the HR dataset.
*   **Power BI Dashboard (`Project 1.pbix`):** An interactive visualization that provides a high-level overview of the HR metrics.

## Tools Used

*   **SQL:** Data cleaning, transformation, and analysis.
*   **Power BI:** Data visualization and dashboard creation.

## Key Features

*   Analysis of employee distribution across departments.
*   Visualization of workforce demographics.
*   Tracking of key HR performance indicators (KPIs).

## How to Use

1.  **SQL:** Import the provided script into your SQL environment to view the data structure.
2.  **Power BI:** Open the `.pbix` file using Power BI Desktop to interact with the dashboard.
